TensorFlow

TensorFlow es una biblioteca de código abierto para aprendizaje automático a través de un rango de tareas, y desarrollado por Google.

Keras

Keras es una biblioteca de Redes Neuronales de Código Abierto. Está especialmente diseñada para posibilitar la experimentación en poco tiempo con redes de Aprendizaje Profundo.

Python

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en una sintaxis que favorezca un código legible.
